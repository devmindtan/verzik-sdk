Trang test ký MetaMask

Chạy trong trình duyệt để có thể mở popup của MetaMask. Node test script không thể tự bật ví, vì không có window.ethereum.

1. Upload draft rồi ký

Chỉ cần file + recipient. Phần hash và payload ký sẽ tự sinh để test popup MetaMask.

Không cần nhập ciphertextHash / encryptionMetaHash nữa. Page sẽ tự sinh để test flow.
Tip: có thể nhập nhiều recipient bằng dấu phẩy. Backend hiện nhận keys.encrypted_key kiểu string, nên SDK sẽ gửi key đầu tiên vào encrypted_key và toàn bộ danh sách vào keys.recipient_keys.
